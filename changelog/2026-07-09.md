## 追加:bootstrap-only 摆动脚接触惩罚(72k step)

新课程 run `logs/rsl_rl/bdx_walk_imitation/2026-07-09_17-02-11/model_700.pt` 诊断:

| 指标 | model_700 |
|---|---|

| 每脚接触时间比例 | **0.995** |

| torso_pos_xy instant reward | 0.0063 |
| path-frame distance p95 | 0.25m clamp |
| 实际离地间隙 | p50 1.6cm / p95 3.3cm |

结论: `rew_k_torso_pos_xy=200` 没有被原地扭动刷到分,因为 path-frame distance 已经顶到 0.25m clamp、
`torso_pos_xy` 均值几乎为 0。仍不抬脚的主要原因是 stage0 里双脚贴地还能拿
`survival + lin_vel_xy 底分 + contact_match 半匹配分`,没有直接惩罚"参考摆动但实际仍接触"。

改动:
- `oceanisaaclab_walk_env_cfg.py`
  - `enable_bootstrap_swing_contact_penalty=True`
  - `bootstrap_swing_contact_penalty_until_step=72_000`
  - `rew_w_bootstrap_swing_contact=-2.0`
- `oceanisaaclab_walk_env.py`
  - 新增 reward/log 项 `bootstrap_swing_contact`;
  - 当 `ref feet_contact=0` 且实际脚仍接触时扣分:
    `-2.0 * Σ I[reference swing and actual contact]`;
  - 仅 `common_step_counter < 72_000` 生效,之后自动为 0,恢复论文 reward 主体。

训练建议:从头重训,在 500/1000/2000/3000 iter 诊断。若 1000~2000 iter 接触比例仍接近 1.0,
下一步再考虑把截止延到 `144_000` 或把权重临时加到 `-3.0`;否则保持当前 72k 配置。

## 追加:摆动脚接触惩罚增强到 -6 / 144k

TensorBoard 显示 `Episode_Reward/bootstrap_swing_contact` 很快稳定到接近 `-2`,说明新惩罚项已经生效,
但策略仍选择双脚贴地硬吃惩罚；同时 `contact_match≈1.0`、`lin_vel_xy≈0.8`、`survival=20` 仍提供舒服底分。
因此判断不是惩罚未触发,而是 `-2/72k` 强度不够。

改动:
- `bootstrap_swing_contact_penalty_until_step`: `72_000 -> 144_000`
- `rew_w_bootstrap_swing_contact`: `-2.0 -> -6.0`

验收:
- 若 1000~2000 iter 后每脚接触比例从 `0.99~1.0` 降到 `<0.9`,说明开始破局;
- 若大量摔倒、`Episode_Termination/time_out` 明显下降,再回调到 `-4.0`;
- 若 `bootstrap_swing_contact` 仍长期打满到约 `-6`,说明还在硬吃,下一步应检查动作探索/熵或直接对 swing foot clearance 加 bootstrap 项。

  - `vx=(0.0, 0.25)`, `backward_prob=0.5`
  - `vy=(-0.15, 0.15)`
  - `wz=(-0.8, 0.8)`
  - `stand_prob=0.15`
  - `turn_prob=0.25`
- 保留课程的部分:
  - `bootstrap_swing_contact=-6.0` 仅前 `144_000` common steps 生效;
  - 头命令 stage4 半幅、stage5 满幅;
  - paper disturbance 仍从 stage5 后开始拉满。
- 修正 `turn_in_place` 采样:原地转向样本现在同时置 `vx=0, vy=0`,避免打开侧向命令后
  "turn in place" 混入侧移。

说明:
- `vx=(0.0, 0.25)` 是前后速度的幅值范围,不是只有正向速度;
- 采样时先取正幅值,再对 moving 样本按 `backward_prob=0.5` 将约一半 `vx *= -1.0`,
  因此非静止/非原地转向样本中约一半是 `vx=-0.25~0.0` 后退命令;
- `stand_prob=0.15` 样本为 `vx=vy=wz=0`, `turn_prob=0.25` 样本为 `vx=vy=0,wz≠0`。

预期:模型从一开始同时学习走/站/后退/转向/侧移,只用临时摆动脚接触惩罚防止双脚贴地逃课。

## 当前最终配置快照

### Reward 项和值

所有 reward term 在 `_get_rewards()` 中先按下表权重计算,最后统一乘 `step_dt=0.02`。
`bootstrap_swing_contact` 是临时破局项,仅 `common_step_counter < 144_000` 生效。

| 项 | 权重/参数 | 形式 |
|---|---:|---|
| torso_pos_xy | w=1.0, k=200.0 | `exp(-k * ||p_pf - p_ref||^2)` |
| torso_orient | w=1.0, k=20.0 | `exp(-k * orient_err^2)` |
| lin_vel_xy | w=1.0, k=8.0 | `exp(-k * ||v_xy - v_ref_xy||^2)` |
| lin_vel_z | w=1.0, k=8.0 | `exp(-k * (v_z - v_ref_z)^2)` |
| ang_vel_xy | w=0.5, k=2.0 | `exp(-k * ||ω_xy - ω_ref_xy||^2)` |
| ang_vel_z | w=0.5, k=2.0 | `exp(-k * (ω_z - ω_ref_z)^2)` |
| leg_joint_pos | -15.0 | `-15 * Σ(q_leg - q_ref)^2` |
| leg_joint_vel | -1.0e-3 | `-1e-3 * Σ(qd_leg - qd_ref)^2` |
| contact_match | 1.0 | `Σ I[c_i = c_ref_i]` |
| bootstrap_swing_contact | -6.0 | `-6 * Σ I[c_ref_i=swing 且 c_i=contact]`, 144k 后为 0 |
| torque | -1.0e-3 | `-1e-3 * Στ_leg^2` |
| joint_acc | -2.5e-6 | `-2.5e-6 * Σqdd_leg^2` |
| action_rate | -1.5 | `-1.5 * Σ(a_t - a_{t-1})^2`, 腿动作 |
| action_acc | -0.45 | `-0.45 * Σ(a_t - 2a_{t-1} + a_{t-2})^2`, 腿动作 |
| neck_joint_pos | -100.0 | `-100 * Σ(q_neck - q_neck_ref)^2` |
| neck_joint_vel | -1.0 | `-1 * Σqd_neck^2` |
| neck_action_rate | -5.0 | `-5 * Σ(a_neck_t - a_neck_{t-1})^2` |
| neck_action_acc | -5.0 | `-5 * Σ(a_neck_t - 2a_neck_{t-1} + a_neck_{t-2})^2` |
| survival | 20.0 | `20 * (1 - terminated)` |

- `move_command_threshold=0.08`,小于阈值的 `vx/vy` 会置 0。
